# Consulta de climas
Esta es una aplicacion web sencilla en la cual podemos consultar la temperatura promedio, minima, maxima y humedad de cualquier ciudad.

#### Acá podemos ingresar el nombre de la ciudad y país de la misma
![consulta clima](https://drive.google.com/file/d/1InnK_-pqd2iHoE6q8iaauQDieiF8PSw7/view)

Al precionar en buscar nos deberia de aparecer los datos de la temperatura de esa ciudad.
![respuesta clima](https://drive.google.com/file/d/1hOVpWaSziHdq4biRJ35Wxu5BnZdv-tW3/view?usp=sharing)

## Proceso para inicialiarlo en local
### `npm start`

Inicializamos el proyecto con npm start.
Abrimos [http://localhost:3000](http://localhost:3000) en el navegador en caso de que no se haga automaticamente y asi poder ver la aplicacion corriendo en local.
